# Guía de Implementación y Despliegue

## 1. Configuración en Supabase
- Crear proyecto.
- Crear buckets de almacenamiento: `public-assets` y `medical-files`.
- Obtener credenciales API y string de conexión DB.

## 2. Despliegue en Vercel
- Conectar repositorio GitHub.
- Configurar variables de entorno.
- Desplegar rama `main`.

## 3. Flujo con Cursor AI
- Abrir proyecto localmente en Cursor.
- Referenciar `REQUIREMENTS.md` y `tech-stack.md` para generar código.
